##  Clases Implementadas

### Dish2 (Plato)
Representa un plato del menú del restaurante.

**Atributos:**
- `name` - Nombre del plato
- `category` - Categoría (Plato Fuerte, Sopa, etc.)
- `price` - Precio en pesos colombianos

**Métodos:**
- `showInfo()` - Muestra información del plato

##  Cómo Ejecutar
```bash
# Compilar
javac Dish2.java Main.java

# Ejecutar
java Main
```
